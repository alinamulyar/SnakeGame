SnakeGame

This repository is created for the classic Snake Game project.

Structures description:

Food - a class for a collectible item Field - the game board Direction - the movement of the snake Engine - game engine Point - position on the field Snake - player

The main task of the project is to develop the game, that will include structures to represent the snake, collectable food, game field represented as a grid of cells.

Game Rules: In the classic Snake game, your objective is to control the snake, guiding it to consume as much food as possible while avoiding collisions with walls and its own tail to grow your snake as long as possible and to achieve the highest score.
